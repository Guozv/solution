# P3643

简单暴力的思路

对暴力 dp[i][j] 表示前 i 个学校 j 为最大值的方案数。

$$dp[i][j]=dp[i-1][j]+[a_i\le j \le b_i]\sum_{k=1}^{j-1}dp[i-1][k]$$

容易想到多项式维护这个式子，后面区间中的求和可以使用离散微积分做到快速求表达式。

最后式子是分段函数的形式，注意细节，维护一下常数项即可
